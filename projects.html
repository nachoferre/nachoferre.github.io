Real Time Energy Monitoring

The Real Time Energy Monitoring is the project that I developed for my final thesis before college graduation. It consists of a platform that allows the user to control the energy consumption in DC. It requires a NodeMCU board and a Raspberry Pi or similar. After processing the information, the board shows the data gathered in a customized webpage in freeboard.io. All the information recorded is shared via mqtt mainly because of the speed and reliability of this communication protocol.

The main technologies used for this Project are Lua, Mosca/Mosquitto, Node-Red and Freeboard.io
The project is libre software and can be downloaded in the following link Github
