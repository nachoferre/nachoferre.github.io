Real Time Energy Monitoring

This is the project I developed for my final thesis. It consists in a platform wich allow you to control the energy consumption in DC. It requires a NodeMCU board and a board computer like a Raspberry Pi. It shows the information gathered in a customized webpage provided by freeboard.io. All the information is shared via mqtt, because is reliable and fast.

The main technologies it uses are Lua, Mosca/Mosquitto, Node-Red and Freeboard.io
The project is libre software and can be downloaded in the following link Github
